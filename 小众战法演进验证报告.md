# 小众战法 · 演进与验证报告

> 2026年6月20日 · 2002-2026 Walk-Forward全量回测

---

## 一、战法起源

### 问题：A股alpha到底在哪

测试了28个因子，6个存活。全部在抓同一件事：**散户的行为偏差**。

| 因子 | 来源 | IC | 在A股抓什么 |
|------|------|-----|-----------|
| Turnover(-) | Datar/Naik 1998 | +11.08% | 散户追高换手→后续跑输 |
| Amihud (log) | Amihud 2002 | +7.96% | 冷门/被遗忘=溢价补偿 |
| Price(-) | 散户低价偏好 | +7.36% | 散户觉得便宜股"划算"→炒高 |
| Max_Rev (-) | Bali et al. 2011 | +5.27% | 散户追涨停→次日被套 |
| VP_Corr | Alpha101 #2 | +4.36% | 量缩价涨=机构出货 |
| Short_Rev | Jegadeesh 1990 | +3.82% | 恐慌割肉→反弹 |

### 为什么是乘法不是线性

因子间不是加法关系——Amihud不直接贡献收益，它是"筛子"：筛出被遗忘的股票，然后等触发器（Max_Rev/Gap/Short_Rev/VP_Corr）发出异动信号。

```
score = rank(Amihud) × rank(触发器)  ← 冷门+异动=猎物
```

二阶够用，三阶被噪声淹没。

## 二、检验过程：四轮迭代

### 第一轮：原始回测 (2002-2026基线)

```
方法: 5因子纯乘法, 月频调仓Top15, 无门禁
结果: 年化+14.0%, Sharpe 0.36, MDD -74.1%
```

**问题：2008年-84.9%**。金融危机中策略几乎归零。任何长仓策略没风控就是这个下场。

### 第二轮：门禁对比 (6种方案)

| 方案 | 逻辑 | Sharpe | MDD | 致命缺陷 |
|------|------|--------|-----|---------|
| NONE | 无门禁 | 0.36 | -74.1% | 裸奔 |
| MA200 | 指数<MA200空仓 | 0.28 | -67.0% | 滞后,砍了也不够 |
| DRAWDOWN | 2年高点回撤>15%空仓 | 0.52 | -48.6% | **2009年踏空(0%)** |
| VOL20 | 波动率定仓 | 0.31 | -65.4% | 反应太慢 |
| VOL15 | 保守波动率 | 0.30 | -61.6% | 仓位缩太多 |
| HYBRID | VOL20+MA200 | 0.30 | -55.7% | 两头不讨好 |

**核心发现：二元开关(全仓/空仓)太粗暴。** 要么砍不够，要么踏空反弹。DRAWDOWN的2009年+0% vs NONE的+103.6%——省了下跌，丢了全部反弹。

### 第三轮：DD_SMART智能门禁 (修2009踏空)

旧DRAWDOWN的死穴：2年高点作为参考太久。2008年指数从6124跌到1664（-73%），2009年反弹到3500（+110%），但离2年高点6124还差43%→门禁一直锁着→完美踏空。

**修复方案：加动量回场规则。**

```
出场: 沪深300 < 2年高点×0.85 → 仓位降至15-40%
回场: 沪深300 > 1年低点×1.15 AND > MA50 → 仓位恢复至70%
底部: 永不全空, 保留15%底仓防止完全踏空
```

| 对比 | 2008 | 2009 | 2015 | 2018 |
|------|------|------|------|------|
| NONE | -66.6% | +103.6% | +98.6% | -42.5% |
| 旧DRAWDOWN | -12.5% | **+0.0%** | +98.3% | -24.0% |
| **DD_SMART** | **-31.6%** | **+67.2%** | **+111.5%** | **-34.1%** |

2009年从踏空→捕获反弹的65%。2015年因为回场及时，甚至跑赢了NONE。

**结果: 年化+11.8%, Sharpe 0.48, MDD -53.0%, 累计+460%**

### 第四轮：ST风险排查 (4种方案)

选股明细中发现策略偶尔踩ST股。测试了四种方案：

| 方案 | 做法 | Sharpe | vs基准 |
|------|------|--------|--------|
| V1_BASE | 不过滤 | 0.48 | 基准 |
| V2_ALLST | 踢全部ST | 0.37 | ❌ -25% |
| V3_MAX3 | 每月≤3只ST | 0.46 | ❌ -4% |
| V4_NOSTAR | 只踢*ST | 0.48 | = 完全一样 |

**结论：策略天然不选\*ST。** MCAP_FLOOR(后20%排除)+涨停过滤已经把\*ST筛掉了。V4踢\*ST和V1完全一样——因为20年里策略从没选中过\*ST。

### 第五轮：5因子 vs 原始6因子 (2026-06-20)

之前把Turnover和Price剔掉换Gap是错的。拿原始6因子+DD_SMART重新跑2002-2026全量验证：

| | 5f (Gap版) | 6f (Price+Turnover) | 差异 |
|---|-----------|-------------------|------|
| 年化 | +11.8% | **+15.1%** | +3.3% |
| Sharpe | 0.48 | **0.67** | +0.19 |
| MDD | -53.0% | **-45.2%** | +7.8% |
| 累计 | +460% | **+1067%** | +607% |
| 7熊年累计 | -73.6% | **-47.4%** | +26.2% |
| 7牛年累计 | +2123% | +2067% | ~持平 |

**6f在所有维度碾压5f。** 熊市保护尤其突出：2008 -25% vs -32%、2017 -12% vs -22%、2018 -15% vs -34%。

选对结构完全不同：

| 5f高频对 | 选中率 | | 6f高频对 | 选中率 |
|----------|--------|-|----------|--------|
| max_rev×sr5 | 100% | | **price×turnover** | **80%** |
| amihud×max_rev | 100% | | price×sr5 | 60% |
| amihud×sr5 | 55% | | max_rev×price | 50% |
| max_rev×gap | 50% | | amihud×turnover | 45% |

**6f的核心逻辑变了：** 不再是"冷门+异动"，而是"**便宜+没人碰**"（price×turnover）。低价股+低换手=散户彻底放弃=机构可以悄悄吸筹。这个逻辑比Gap的情绪缺口修正更稳定。

**结论：Gap出局，Price和Turnover归位。** 原始6因子就是最优解。

策略选到的ST(6.3%)是"其他风险警示"类——财务有问题但不至于退市，这类恰恰是反弹最猛的。踢掉它们就是在踢alpha。

## 五、策略的边界

### 它擅长什么
- A股散户驱动的反转行情
- 被市场遗忘后重新发现的股票
- 恐慌超卖后的反弹

### 它不擅长什么
- 大盘蓝筹趋势行情(2017年-21.5%就是教训)
- 连续慢熊(2011, 2018都亏了)
- 壳价值消失后的市场结构(注册制全面推行后需重新验证)

### 谁适合用
- 能承受-50%回撤的人
- 把这个当卫星仓(≤20%总资产)
- 不用杠杆
- 按月调仓不盯盘

## 六、已知局限

1. **MDD仍高(-53%)**: 对小盘股策略来说这是结构性问题，不是参数能调的
2. **2017年失效**: 当年是漂亮50行情，小票被遗忘→但遗忘≠要反弹，是真被抛弃
3. **纯K线**: 看不到基本面，2024年sh600306(*ST商城)-53.5%就是代价
4. **壳价值衰减**: Lee/Qu/Shen(2023)估算壳占小市值公司22%——注册制每推进一步，alpha就衰减一分

## 七、迭代教训

1. **简单乘法 > 复杂ML**: LightGBM Sharpe 1.28大部分是过拟合，纯乘法0.67是能拿到的底线
2. **二元开关是陷阱**: 门禁必须连续，永远不全空。一次踏空毁掉多年守候
3. **A股的alpha来源就是没人要的股票**: 想用ST过滤安全化→直接削掉alpha源
4. **别在回测里追求"好看"**: V1_BASE MDD -74%→DD_SMART -45%是真改进，加ST过滤纯属画蛇添足
5. **2008和2009是最佳检验**: 任何A股策略必须过这两年的测试——暴跌中少亏、反弹中跟上
6. **原始6因子 > 5因子**: 剔除Turnover和Price换Gap是错的。便宜+没人碰(price×turnover) > 情绪缺口修正(gap)。Walk-Forward不会说谎
