## 一、战法起源：28因子→6因子

### 1.1 A股底层逻辑

A股是**融资市**，不是投资市。散户交易量占80%+，股票是筹码不是所有权。

```
原始股东(成本1毛) → IPO套现 → 机构/游资(成本10块) → 拉升出货 → 散户(成本30块)
                                                                         ↓
                                                                  割肉离场
                                                                         ↓
                                                              机构再次吸筹...
```

**6篇顶刊学术基石：**
- Allen/Qian/Qian (2005, JFE): 上市公司≠经济核心
- Carpenter/Lu/Whitelaw (2021, JFE): 价格信息含量追上美股但国企拖后腿
- Lee/Qu/Shen (2023, Review of Finance): 壳价值占小公司市值22%
- Liu/Stambaugh/Yuan (2019, JFE): 剔除壳后中国版三因子才有效
- Jiang/Lee/Yue (2010, JFE): 大股东掏空上市公司
- Allen/Qian/Shan/Zhu (2024, JF): A股长期跑输GDP 15%

### 1.2 28因子筛选过程

```
28因子测试 → IC检验 → 相关性矩阵去重 → 6因子存活
  ↓
Turnover(-) IC=+11.08% | Amihud(log) IC=+7.96% | Price(-) IC=+7.36%
Max_Ret(-) IC=+5.27%   | VP_Corr IC=+4.36%     | Short_Rev IC=+3.82%
```

**全部在抓同一件事：散户的行为偏差。** 不是选好公司，是选被散户错误定价的股票。

### 1.3 为什么是乘法不是线性

```
score = rank(Amihud) × rank(触发器)  ← 冷门+异动=猎物
```

- Amihud是"筛子"：筛出被遗忘的冷门股
- Turnover/MaxRev/SR5是"触发器"：等异动信号
- 二阶乘法放大信号，三阶被噪声淹没
- DeMiguel(2009)复现：简单方法稳定跑赢复杂ML

### 2.2 核心选对：Price×Turnover (80%选中率)

```
6f高频对:  Price×Turnover 80% | Price×SR5 60% | MaxRev×Price 50% | Amihud×Turnover 45%
5f高频对:  MaxRev×SR5 100%    | Amihud×MaxRev 100%
```

**"便宜+没人碰" > "情绪缺口修正"。** Gap因子出局，Price和Turnover归位。

### 2.3 Fama-MacBeth统计验证

| 检验对象 | NW t值 | p值 | 结论 |
|---------|--------|-----|------|
| Price (单因子) | +0.64 | 0.521 | ❌ 不显著 |
| Turnover (单因子) | -1.05 | 0.294 | ❌ 不显著 |
| **Price×Turnover** | **+4.16** | **0.000** | ✅ 高度显著 |
| Amihud (单因子) | +1.77 | 0.078 | ⚠ 边缘 |
| MaxRev (单因子) | -1.34 | 0.182 | ❌ 不显著 |
| **Amihud×MaxRev** | **+2.61** | **0.009** | ✅ 显著 |

**核心发现：1+1=0，但1×1=4。** 单因子都不显著，乘法交互高度显著——这是整个战法的统计基础。

### 3.2 v1到v2的关键改进：36组网格搜索

```
v1参数: exit=-20%  reentry=+15%  floor=15%
v2参数: exit=-12%  reentry=+10%  floor=10%

核心变化:
  - 更早出场(-12%→-20%): 危机萌芽时就减仓,不等深跌
  - 更快回场(+10%→+15%): 反弹启动就加仓,不踏空
  - 更低底仓(10%→15%): 危机中保留更少风险
```

### 3.4 门禁基准：必须是沪深300

**HS300是大盘的领先指标**——大资金先跑大票，小票随后崩。用HS300等于提前拉警报。

## 七、迭代教训

1. **简单乘法 > 复杂ML** — LightGBM Sharpe 1.28大部分过拟合，纯乘法0.70是真实alpha
2. **二元开关是陷阱** — 门禁必须连续，永远不全空
3. **HS300是领先指标** — 大票先跌小票后崩，别用小票体温计
4. **ST是alpha源不是风险** — 过滤ST=自断alpha，策略天然避*ST
5. **Alpha=小盘暴露** — 想用市值中性化剥离=策略死亡
6. **原始6因子 > 5因子** — Price×Turnover(便宜+没人碰)碾压Gap(情绪缺口)
7. **交互>单因子** — FM证明：单因子t<1，交互t>4，差异巨大
8. **2008+2009是最佳检验** — 暴跌中少亏、反弹中跟上
